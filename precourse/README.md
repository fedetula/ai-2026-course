# Precourse Materials

Complete these before Session 1. Start with the setup, then work through the prerequisites.

## Setup

1. Follow the instructions in [SETUP.md](../SETUP.md) to install Docker and start the environment
2. Run `test_setup.ipynb` to verify everything works

## Hard Prerequisites (Required)

You must be comfortable with these **before** the course starts. These notebooks are refreshers, not tutorials from zero. If you struggle with them, review the references linked in each notebook.

| Notebook | Topic | Estimated Time |
|----------|-------|---------------|
| [python.ipynb](hard-prerequisites/python.ipynb) | Functions, classes, comprehensions, file I/O, error handling | 30-45 min |
| [numpy.ipynb](hard-prerequisites/numpy.ipynb) | Arrays, indexing, broadcasting, vectorized operations | 30-45 min |
| [linear-algebra.ipynb](hard-prerequisites/linear-algebra.ipynb) | Vectors, matrices, dot product, matrix multiplication | 30-45 min |
| [calculus.ipynb](hard-prerequisites/calculus.ipynb) | Derivatives, chain rule, partial derivatives | 30-45 min |
| [probability-statistics.ipynb](hard-prerequisites/probability-statistics.ipynb) | Mean, variance, distributions, Bayes' theorem | 30-45 min |
