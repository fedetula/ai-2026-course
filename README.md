## Quick Start

1. Install [Docker Desktop](https://www.docker.com/products/docker-desktop/) (Windows/Mac) or Docker Engine (Linux)
2. Clone this repo and start:

```bash
git clone https://github.com/unrn-ai-2026/ai-2026-course.git
cd ai-2026-course
docker compose up
```

3. Open http://localhost:8888 in your browser
4. Open `precourse/test_setup.ipynb` and run all cells to verify your setup
5. Work through the [precourse materials](precourse/README.md) before Session 1

See [SETUP.md](SETUP.md) for detailed instructions and troubleshooting.
